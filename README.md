# Dual-Mode Text Summarization System using NLP

An NLP-based system that generates extractive or abstractive summaries from long text using classical and deep learning techniques.

## Features
- Extractive text summarization
- Abstractive text summarization
- User-selectable summarization mode
- Simple and interactive interface

## Technologies
- Python
- NLP
- NLTK
- Transformers
- Streamlit
